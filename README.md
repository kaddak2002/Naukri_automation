# Naukri_automation
Automated the Naukri Login, update the profile, job search, and job application handling with dynamic filters using playwright with javaScript
Developed an end-to-end job application automation framework for Naukri.com using Playwright with JavaScript and Page Object Model (POM) design pattern.
Implemented automated workflows for login, resume headline updates, job search, and job application handling with dynamic filters such as designation, location, and work mode (Remote/Hybrid/WFO).
Designed reusable utility classes, fixtures, and page components following BDD automation framework architecture to improve scalability, maintainability, and code reusability.
Automated intelligent job application flow by detecting direct “Apply” options, skipping already applied jobs, and handling external company career portal redirections for manual intervention.
